The table on this page contains all of the Questions and Notes (Q/N) ever entered.  Each Q/N has a location which represents its progress from being start to either Published or sent to Trash.  The table can then be filtered to show a single location, as indicated in the Presets.  For more information on how to use the table, see Navigating Common Tables.  This topic describes the contents of the Edit page starting from the top.  Instructions on how to use this page can be found in Instructions for Note Takers and Instructions for Directors.
Email Staff link
The Email staff link will create an email from you to all the Rehearsal Note (RN) staff.  In addition, almost every name displayed on this page is a link to create an email to send to that person.
Online Singers & Staff links
This page is designed to be collaborative with anyone who is currently active on both the public Rehearsal Notes page and this page. 
  • "ONLINE Singers: " will show a list of anyone who currently is active on the public page.
    • If you are currently publishing notes or need to modify an already published note, this list gives you the ability to contact these people, if necessary.
      • Issues about modifying published notes can be found in the "Done" column below and in Note Taker's Add/Edit Notes.
    • Changes you make to the location "Published" will appear immediately on the public page.  See Singer's Rehearsal Notes Page for what they see.
  • "Staff: " shows a list of all RN staff. 
    • Those that are currently active on this page will be outlined in blue .
    • Any changes they make will appear immediately on this page, also briefly outlined in blue .
Presets drop down
  • All Notes: Shows the entire list of Q/N sorted by Song and Start Measure (ms).
  • Note Taker's Inbox: This is the default setting for Note Takers., showing only Questions with location = "Note Taker's Inbox"
    • It is the target location for Questions asked by Singers.
  • Director's Inboxes: There are multiple Director Inboxes, showing Questions with location = "Director's Inbox" .
    • It is the target location for Questions from Note Takers and those forwarded from Singers.
    • The first, "Director's Inbox", is for all contents shown together.
    • The additional presets are for each individual Director.
  • For Review: Shows all Notes that are ready for review and publishing, with location = "For Review"
  • Not Published: Provides a quick way to see if there are any pending actions that need to be taken.
    • It shows all Q/N in the Note Taker's Inbox, Director's Inboxes, and For Review.
  • Published: Shows all notes with location = "Published"
    • This list is identical to what the Singers see on the public Rehearsal Notes page.
  • Trash: Shows all notes with location = "Trash".
    • Nothing is ever deleted, so that we keep a complete record of all decisions and actions taken for all Q/N.
    • Q/N ultimately end up either Published or in Trash.
    • When a Q/N is sent to Trash, a dialog is shown to fill in the reason why, and the reason is placed in the DISCUSSION column.
Ask Question button
The  ASK QUESTION button brings up the Ask Question form and enables Note Takers to send questions to the Directors.  These questions get set to location = "Director's Inbox".
Add Note button
The ADD NOTE button brings up the Add Note form and is used to enter Notes which were recorded during Workshops or provided by Directors.  These notes get set to location = "For Review".  There they should be reviewed by either the RN Administrator or another Note Taker before being published.

table Columns

Clicking anywhere on a Row, except in the Location, Author, Editors, and Actions columns, will bring up the Edit Q/N form for that Q/N.

  • Done - For Published notes, this column indicates how many Singers have marked the note "Done"
    • This means that the note has already been copied into someone's music
    • Editing an already Published note with at least one "Done" mark is very limited.
      • Instructions on how to edit these notes are in the Edit Notes form.
  • Song - What song the Q/N refers to.  Use this column's filter to work on one song at a time.
  • ms - Starting measure of each Q/N. 
    • To filter on this column, enter a single measure number and it will include +/- 10 measures.  Enter 0 to see notes for the entire song.
  • Parts - the Voice Parts that the Q/N pertains to.
    • Generally only S,A,T,B can be used in this filter and they use an AND logic.
    • In other words, a filter of SA will only show Q/Ns that pertain to both S and A.
    • There is no option for one filter that will include Q/Ns that pertain to either S or A.
  • Question/Note - the text of the Q/N.  This is the text that is Published and Singers will see.
  • Location - shows the current location, or progress state, of the Q/N
    • Clicking on a cell in this column will bring up a quick Location change drop down.
    • The drop down will only show the possible location options, based on the Q/N's current location.
  • Dir - shows the initials of the Director responsible for this Q/N. 
    • Usually this is the default Director indicated in the Song List Settings, but this can be overridden in the edit forms,
  • Author - This is the name of the original person who authored/started, the Question or Note.
    • Clicking on the name will start an email to that person.
  • Editors - shows a list of everyone who has edited or reviewed the Q/N since it was created.
    • The most recent Editor will be shown in slightly larger print.
    • Clicking on any of the names will start an email to that person.
  • Date,Time - shows the time of the most recent modification to the Q/N.
  • Discussion - This is a private field, only visible to the RN staff and is never Published.
    • It provides a place for RN staff to ask questions or comment on a Q/N.
    • When a Q/N is sent to Trash, the reason is entered here.
  • Actions - This is the right-most column (with no header name), and contains icons described above the table to the right:
Duplicate - Brings up the Add Note form pre-filled with a copy of the current Note. 
This can be useful when the same note must be created for a set of measures, or is for both choirs of a song that has two entries in the Song List, for Choir 1 and Choir 2.

History - Brings up the History Log for the Q/N. 
This is a recording of the entry and every modification of the Q/N.
Sometimes it can help to understand who processed a note, when, and how it ended up where it is.

Highlight - In some other Common Tables, it is possible to click on a row to  highlight  it. 
In this table, clicking a row brings up the Edit form, so this action button is provided, as an alternative, to do the highlighting.  It is useful when changing the Preset, filtering, or sorting.  It marks row(s) so that they can be found after the change. 
There are some actions that automatically highlight rows.  For example, if you use the Location quick drop-down to change a Q/N to a location outside of the current filter, it will be hidden (but also highlighted).  You can then change the filter and find it, because it will remain highlighted. 
Clicking this action button enables you to manually select which row(s) to highlight.

Permanently Delete - This is only available to the RN Administrator.
It only appears for those Q/N already set to location = "Trash".  It serves only as a cleanup option.  In the rare case when a Q/N was really a mistake and there is no point in keeping it in the historical records, it can be completely removed using this action button.  This action cannot be undone.
