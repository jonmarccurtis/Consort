Note Takers are those who have volunteered to record changes in the music given by the Directors at Workshops.  These are the instructions on how to enter those "rehearsal notes" online.
Edit Rehearsal notes page

Location: Admin -> Edit Rehearsal Notes
Direct Link: https://www.consortchorale.org/rehearsal-notes-admin
If you do not see this menu item, ask the RN Administrator to add you to the Note Takers list.

This page contains all rehearsal notes and questions in a single table.  Each Q/N has a location, which also defines a work-flow order:
Location Contains Process
Note Taker's Inbox
 Questions asked by Singers
 Either a Note Taker or the Administrator screens the questions, answers them, or forwards them to the Directors.
Director's Inbox
 Questions from Note Takers
 A Director answers the questions and sends them to For Review.
This is the default "Save" location for your Ask Question form.
For Review
 Notes entered by Note Takers
Questions answered by Directors
 Reviewed by the Administrator or other Note Takers, then Published.
This is the default "Save" location for newly added notes.
Published
 Notes to be marked in the scores
 Singers see these notes on the Singer Info -> Rehearsal Notes page, and are to mark them in their music.
Trash
 Duplicates
Questions answered without needing a Note
Other errors
 Q/Ns are never completely erased, so that we can have a history about all the decisions
 that have been made and discussed about songs/measures.

You can use the "Presets" drop-down at the top of the page, to filter the table on the above locations.

SUGGESTED WORKFLOW
  1. Set Presets to "All Notes".
  2. (optional) Filter to the Song you are working on by entering its name in the filter at the top of the "Song" column.
  3. Check for Duplicates
    1. The notes are sorted by start measure (ms) by default.
    2. If you put a measure number into the "ms" filter, it will show +/- 10 measures.
    3. You can click the clock icon on the right of any note to see its complete history.
    4. If you find a duplicate already entered, set its location = "Trash".
    5. You can email a note's last Editor by clicking on their name.
  4. Use ADD NOTE or ASK QUESTION buttons to enter your information.
    1. If asking a question from another singer, select their name in the "On behalf of" drop-down, so they can find the question in their Track Questions form.
Additional Information
  • If any other RN staff are currently online, their name(s) will be outlined in blue, and any changes they make will appear immediately on your page, briefly outlined in blue.
  • Notes you enter are set to location = For Review.  The ideal process is that each note will be reviewed by another Note Taker or the Administrator before going out to the singers, which is location = Published.
  • Notes and Questions can be modified at any time, unless they have been Published and at least one Singer has marked them "Done".  Then, further editing is limited, and should only be done in a way that those singers who have already recorded the markings in their music will know what needs to now be changed.
Suggestions when creating notes
[seg-rnote-format]
Not all Questions need to become Notes, or forwarded to Directors
If a question can be answered without needing to make a change/marking in the music:
  1. Email the Singer asking the question, sending them the answer
    1. Click on the clock icon to the right of the question
    2. In the first revision, click on the "Author" name to start an email to them
  2. Set the question's location = Trash
    1. In the Trash form, add the answer sent to the singer.
    2. This keeps a log of the answer in case it is asked again.
When a song has 2 choirs (Choir1 and Choir2)
These songs will have 2 entries in the Song list.  If a question/note refers to both choirs:
  1. Add it to one of the choir's song entries.
  2. Click on the Duplicate icon in the right-hand column.
  3. Change the duplicate to be for the other choir.

Note refers to the entire song
Leave the (start) measure blank.

Note refers to a range of measures (12-16)
Only enter the starting measure.  "Measure" is only used for sorting the notes in the table.  The full range of measures, or the ending measure, can be indicated in the note's text.

Note refers to a non-contiguous set of measures (12,18,25)
  1. Enter it for the first measure
  2. Duplicate for the additional measures
