The Song List is the beginning of the Rehearsal Notes settings.  The only information it is dependent upon is the list of Directors.  The list of Directors should not be changed once the Song List has been entered, but see Singers Settings form if it does.  Everything else is dependent on the Song List.  Preferably it should never be changed once any Singer's Voice Parts have been assigned or any Rehearsal Notes have been created.

WARNING: If changes are needed after Voice Parts have been assigned or Rehearsal Notes added, see the detailed instructions under CHANGING THE SONG LIST below.

Delete button

This is the first column, but is only available for Songs which do not have any dependent Rehearsal Notes.  See CHANGING THE SONG LIST for details.

Measures

The two boxes are for start and end measures.  Normally start measure is 1, but allows for when a score's numbering does not start at 1.  These numbers are used to validate the starting measure of Rehearsal Notes.

Voice Parts
  • Use the same voice parts as in the score.  For example, if the score is SAA, use "S,A1,A2".  If the Directors want the voice parts to be split High/Middle/Low, then on the Singer List assign High => S, Middle => A1 and Low => A2. 
  • Only one set of voice parts can be assigned to each song.  If the song has more complex voice splits in different places, choose the most common for the overall song, and then call out exceptions on individual Rehearsal Notes.  Another option is create a Rehearsal Note that outlines the complex splits.  Sometimes its a good idea to do both.
  • Each voice part must be 1-2 characters in length, and unique.  If the song has Middle voices for both women and men, use "um" and "lm" for Upper and Lower Middle voices.
  • Voice Parts can use any letter designation that fits with the score.  So, for example a twelve part Vaughn-Williams piece done in Opus 26 used "S1,Sm,S2,A1,Am,A2,T1,Tm,T2,B1,Bm,B2".
  • Voice Parts are displayed in the order they are listed on the Song List.  They should be listed high-to-low.

For a further explanation of how Voice Parts are used by Singers, see Singers Voice Parts form.

Download CSV

The button is provided as a way to backup the settings, and could be used as way for Directors to review the settings and perhaps send back changes.  There is no "Import CSV", as this would bypass many validation checks in the online form, and could introduce incorrect settings.

Changing the Song List

The settings in the song list are used throughout the RN system.  All of its other settings as well as the Notes themselves depend upon this list.  So making changes to the list after Voice Parts have been assigned or Rehearsal Notes have been written is possible, but should be done very carefully.  One way to do it safely is to go to the Administrator Tab and delete all Rehearsal Notes.  But this may not be desired or necessary.  Here is a list of changes that can be done without clearing the entire Notes database.

Before making these changes, it is best to save current copies of the RN settings and data as described in Settings, Administrator tab.

Change After change ... Details
Adding a Song
 Make sure Singer List VP assignments are OK.
 After adding the Song, go to the Singer List tab and make sure that the default voice part assignments were applied to the new song for each Singer.
Removing a Song
 Only allowed if the Song has no Notes assigned to it.
 The Song List will not allow a Song to be removed if it has any Rehearsal Notes.  To remove a Song with Notes, go to the Rehearsal Notes Edit Page, move all the notes for this song to Trash, and then have the RN Administrator permanently delete them.  Then the delete button will become available for the Song and it can be removed.
Changing a Song's Name
 Automatically updates everywhere.
 Fortunately, the Song's name is not stored anywhere else in the RN data.  If you change its name here, it will be changed everywhere: in the Voice Part Assignments, Singer Voice Parts, Rehearsal Notes, and in note Histories.  Technically the history will be "rewritten" as if that was the name all the time.  But the change itself is harmless.
Changing Start/End Measures
 Notes outside the new limits need to be adjusted.
 This is a relatively safe change.  The only time these measure limits are used are when the Add/Edit form is saved.  For example: If a note already exists, starting at measure 75, and the End Measure is reset to 70, that note will continue to say it starts at m75.  However, if it is edited, it will not allow the edit to be Saved until its start measure is changed to be less than 71.
Change Director Does not affect existing Questions, so may need to update them. This setting is only applied as the default Director for new Questions (and Notes).  It is only used to sort Questions in the Director's Inbox.  Since changing it does not affect existing Questions, they may need to be manually reset. For completely adding/removing a Director from the list, see Settings, Singer List.
Adding a Voice Part
 Check Singer List for changes.  Check Notes - particularly those that were for "(all)" voice parts.
 Adding a Voice Part is safe, but check the Singer List to make sure its VP settings are still OK.  Voice Part Assignments are made 2 ways:
  1. Default settings: these are automatically applied when they can be determined.  For example, a T1 will have a default setting T when it is available in a Song.
  2. Overrides: when a VP setting has been changed on the Singer List, it is an override of the default setting.
So, when a new VP is added to a Song, if a Singer's VP was using a default setting, it may be reset to the new VP if possible.  But any Overrides will be unaffected.


Its a bit different for Rehearsal Notes.  The Note's VPs are stored in the note, so this change is not applied to existing Notes.  Check for Notes that were (all).  (all) is not an actual VP setting.  It is automatically shown when every VP has been set for a Song.  Now that a new VP has been added, those songs that showed (all) will now show just the previous list of VPs.  You may need to add the new VP to each of these Notes, if they should still apply to (all) VPs.
Removing a Voice Part
 Need to manually remove the VP from non-default Singer List VP Assignments and in Notes with this VP.
 This has a similar affect in the other direction.  Singers who were using the removed VP as a default setting (#1 above) should get reset to another existing VP, if possible.  But those who had VP Overrides, even if the Override included the removed VP, will be unchanged, and continue to show the remove VP.  If you modify their VP assignments, the removed VP will no longer be an option and will be removed on Save.  So after a remove - check and adjust the settings in Singer List.

Same for Rehearsal Notes.  Notes for that Voice Part will be unchanged.  You should go through those notes, edit and save them, to remove the old VP.
Changing a Voice Part
 Same as Add/Remove.  Recommend doing it that way, one at a time.
 There is no connection between the old and new VP.  This is identical to doing a Remove followed by an Add (or the other way around), so both of the above instructions will need to be applied.  In fact, it is probably wiser to them separately one at a time.
