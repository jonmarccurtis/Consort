This page is only accessible by RN Administrators.

The purpose of this page is delete and reset the Rehearsal Note data, normally done only at the end of one season, getting ready for the next.
Saving Rehearsal Note Data
Before deleting data, and at other times, it is a good idea to save the data.  This is not a "backup", in that there is no way to restore the data from the spreadsheets that are created.   But it does provide a way to record the contents of the tables.  There are 4 tables that can be saved as CSV files:
Table Location How to download
Song List
 Rehearsal Notes Settings -> Song List tab (Direct Link) Click  DOWNLOAD  button
Singers List
 Rehearsal Notes Settings -> Singers List tab (Direct Link) Click  DOWNLOAD  button
Reports
 Rehearsal Notes Settings -> Reporting tab (Direct Link) Click  DOWNLOAD  button
Rehearsal Notes
 Admin -> Edit Rehearsal Notes (Direct Link) Set Presets = "All notes", Click  DOWNLOAD  button
(The only RN data not savable this way, is the History of notes.  See below under Advanced Data Access for where the RN data is stored in the database, and how to access it.)
Select what to delete
Since the RN data has dependencies, dependent data must always be deleted.  This selection process guarantees that. Each choice will show all the data that will be deleted under "Sections to be deleted".
Deleting Rehearsal Notes data on this page is permanent and cannot be recovered.
