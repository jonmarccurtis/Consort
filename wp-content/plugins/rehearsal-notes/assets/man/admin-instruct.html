The primary tasks of the RN Administrator are:
  1. Set up and Maintain the Rehearsal Note system (covered in these instructions)
  2. Review and Publish Rehearsals Notes entered by Note Takers (see "Instructions for Note Takers")
  3. Monitor Questions to get them answered, reviewed, and published (see "Instructions for Directors")

How to set up Rehearsal Notes AT THE BEGINNING OF  A SEASON
There are dependencies which determine the order in which set up should be done:
  1. Setup the list of Artistic Directors
    1. Artistic Directors must have a WordPress role of either Web Assistant or Board.
      1. Go to Admin -> Dashboard -> Users (Direct Link), find their Profile
      2. Set Role to "Web Assistant" (or leave as "Board" if they are a Board Member)
      3. Check Position: Artistic Director
    2. Import them into the RN system:
      1. Go to Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
      2. Click the "Import/Update Singer List" link at the top
    3. (If you did not do step 1.3) Click on their name to open their settings, and check RNotes Positions: Dir.
  2. Setup the list of Songs
    1. Go to Admin -> Dashboard -> Settings -> Rehearsal Notes, Song List Tab (Direct Link)
    2. Verify the Director(s) names are correct
    3. Fill in the songs, following the instructions in Settings, Song List tab.
Because of dependencies:
The list of Directors should not be changed once Songs have been entered in the Song List.
The list of Songs should not be changed once Rehearsal Notes have been created. 
However, most changes are still possible if done carefully.  Instructions for making these changes can be found in Settings, Song List tab.
ONGOING REHEARSAL NOTES SETTINGS
  • Adding someone to the Singers List. When a member registers for the current season:
    • Find their WordPress Profile under: Admin -> Dashboard -> Users (Direct Link)
    • If needed, change their WP Role from "Member" to one of: Singer, Web Assistant, or Board
      • Web Assistant is only needed if they are to be a Note Taker, or have a volunteer task that requires access to the Dashboard.
      • Because of the hierarchy of roles, Web Assistants and Board members are automatically included as Singers.
    • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
    • Click on "Import/Update Singer List"
    • If necessary, click on their name and set their Primary Voice Part, Positions, and adjust any Song Voice Parts settings.

Note: Rehearsal Note's "Singer" position is not the same as the WordPress "Singer" role in a user's Profile.  The WP Singer role is an indication that this person is active in the current season.  WP Roles are also hierarchical - meaning that Web Assistants and Board members are also Singers.  In RN, the Singer position means the the person is actually singing in one or more songs.  Thus, the Accompanists, Reception Coordinator, and other volunteers active in the season will appear in the Singer List, but would not have their "Singer" position checked.

  • Remove someone from Singers List.  When someone drops out of the current season:
    • If their current WP Role is "Singer":
      • Find their WordPress Profile under: Admin -> Dashboard -> Users (Direct Link)
      • Change their WP Role to "Member".
      • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
      • Click on "Import/Update Singer List"  (This will remove them from the list.)
    • If their current/new WP Role is "Web Assistant" or "Board" (they cannot be removed from the Singer List):
      • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
      • Click on their name an uncheck all of their RNotes positions
  • Adding someone to list of Note Takers.  This also grants them access to the Rehearsal Notes Edit page:
    • If they are already a "Web Assistant" or "Board":
      • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
      • Click on their name and check  "NT" in RNotes positions.
    • Else:
      • Go to: Admin -> Dashboard -> Users (Direct Link), and find their Profile
      • Set their WP Role to "Web Assistant"
      • Check "Note Taker" under "Positions" setting.
      • Save their Profile
  • Removing someone from list of Note Takers.  This does not alter any notes they have already entered:
    • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
    • Click on their name and uncheck "NT" in RNotes positions.
    • If needed:
      • Go to: Admin -> Dashboard -> Users (Direct Link), and find their Profile
      • Set their WP Role to "Singer"
      • (You can also skip the Singer Tab and uncheck "Note Taker" in their "Positions" setting here.)
  • Changing a Singer's Primary Voice Part.  This setting is used to sort singer lists, and as a default for Voice Part assignments:
    • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
    • Click on their name and use the Primary voice part dropdown to change the setting.
    • The system will try to keep any specific song voice part assignments, but it is best to check over the list to make sure these are still correct.
  • Changing Voice Part Assignments.  This is a common task, as the Directors work out who is singing each part on each song:
    • Go to: Admin -> Dashboard -> Settings -> Rehearsal Notes, Singer List tab (Direct Link)
    • Here are some approaches:
      • Making only one or two changes
        • Click on the single voice part assignment cell
        • This brings up a tiny dialog to change only that one setting.
      • Changing one Singer's Voice Parts for several Songs
        • Click on their name and use the Singer Settings form
      • Changing one Song's settings for several Singers:
        • Optional - use  HIDE COLUMNS  (see Common Table instructions) to show only the song being edited
        • Click on each assignment to bring up the tiny dialog and make the change
    • Note that these changes on the Singer List are saved immediately.  There is no "Save Changes" button that needs to be clicked when done.
  • Editing/Deleting a Published Rehearsal Note, already copied.  Once a Note has been read (Done count > 0), editing is limited and deleting is not allowed.
    • The edit form will show a warning: You are editing a Published Note that has been read/copied.  Be careful changing these settings ...
    • Admins can bypass the limitations that Note Takers have:
      • First consider the editing options listed in Note Taker's Add/Edit form.
      • Make sure the changes you make do not leave Singers with incorrect markings in their scores.
    • If you need to delete a copied Published Note:
      • Edit the Note, make a small change and Save it.
      • In the next dialog, choose OK to clear everyone's "Done" status.
      • Now it will allow you to send the Note to Trash.
