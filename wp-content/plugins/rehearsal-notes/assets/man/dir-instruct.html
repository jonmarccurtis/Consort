The primary task for Directors is to answer Questions posted in their Inbox.  Directors can also directly enter Rehearsal Notes to avoid having to give them out verbally during rehearsals.

Director's Inbox

Location: Admin -> Edit Rehearsal Notes
Direct Link: https://www.consortchorale.org/rehearsal-notes-admin
The table on this page contains all Questions and Rehearsal Notes, and is filtered using the "Presets" drop down at the top of the page.  The Preset filter should default to your own Inbox.

Answering a question
Click a question to bring up the Edit Form.  Typically you should not need to change any of the top settings, only the question itself.  Here are some suggestions on various kinds of responses:
  • If the answer to the question is a marking in the music, replace the question text with the rehearsal note text.
  • Add an answer following the text of the question.  (A reviewer will rewrite it as a rehearsal note, if needed.)
  • If you need to ask something of the person who sent the question, put that into the bottom box "DISCUSSION".  Text in here is only for the RN Staff and is never included in published Notes.

When done, click .  The Question will become a Note and its location will be set to "For Review".  If you need to edit it further, change your Preset filter to "For Review" or "All notes" to find it.

Adding Notes Directly

Directors have an ADD NOTE button at the top of their Inbox.  Review the "Instructions for Note Takers" for details on how to add notes.
