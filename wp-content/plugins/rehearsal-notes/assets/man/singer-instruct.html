Consort has the tradition of not asking questions about the music during rehearsals, and limiting the amount of time spent in rehearsals going over changes to the music. Instead, changes to the music are posted on this page of the website and it is each singer’s responsibility to copy those changes to their music.

“Done” checkbox:
It's important that you use this checkbox for several reasons:
  • Notes are added continuously throughout the season. When you check off what you’ve already added to your music, you can quickly identify which notes are new, and needed to copied.
  • If a note you have already marked in your music must be changed, your “Done” checkmark will be removed so that you know to update that marking.
  • Marking a note “Done” lets Note Takers know that they must limit what they can change in the note.
Presets dropdown:
This defaults to showing only "My Notes", but most of the time you may want to use "- MY NOTES, NOT DONE (Rec)", which will show only your notes that you haven’t yet marked in your music.

“My notes” and “Parts” Columns
Because the voice parts can be complex (up to 12 in some songs), the "My notes" column has been added to indicate which notes are for your voice parts.

View Voice Parts, Adjust “My note” filter
On the page above the table, click "View Voice Parts, Adjust "My note" filter" to see the voice parts the the Directors have assigned you for each song. You can also use this dialog to set which notes are checked in the “My note” column, and appear in the My Notes filter Preset.

Questions
Use the ASK QUESTION button to send a question to the Note Takers and Directors, and the TRACK QUESTIONS button to see the status of questions you’ve asked. You cannot edit questions after they have been saved, so if you need to make a change, go to the Track Questions form and click on the name of the most recent Editor to send them an email.

Working offline from a printed copy of the Notes
Use the PRINT button to bring up a printable view of the Rehearsal Notes:
  1. Select which notes you want to print. Recommend: Presets = "MY NOTES, NOT DONE (Rec)".
  2. Click the PRINT button.
  3. Use your browser's Print button to print the page.
  4. Be sure to click the button once it is printed!  This marks the printed notes "Done" and ensures that next time "MY NOTES, NOT DONE" will show only changes since the last time you printed.
