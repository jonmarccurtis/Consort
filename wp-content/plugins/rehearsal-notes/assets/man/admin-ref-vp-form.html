Location: Admin -> Dashboard -> Settings -> Rehearsal Notes (second Tab), Click on Singer's name
Direct Link: https://www.consortchorale.org/wp-admin/options-general.php?page=rn-settings&tab=singers

This form enables editing of a Singer's Settings. 
Most of these settings can be changed at any time, but note the cautions below.

Primary voice part
The Primary Voice Part is used to set "default" Voice Part Assignments for each song.  Any changes to Voice Part Assignments (in lower part of the form) are saved as "overrides".  If the Primary Voice Part is changed, only the default assignments will be updated and the overrides will remain in place.  Therefor:
If the Primary Voice Part is changed, make sure that the Voice Part Assignments are also adjusted as needed.
Rnote positions
Three of these settings can also be made in Admin->Dashboard->Settings->Users, User Profile: Positions - as shown in the WP Profile column.
Position Description Changing cautions WP Profile
Singer
 As indicated in the Instructions for Administrators, there is a difference between the WP Role of "Singer" and the RN Position "Singer" in this table: 
  • The Singer List includes all those with WP Role = "Singer". 
  • But on this list, the "Singer" position indicates only those who are actually singing at least one song during the performance.  These singers will be tracked in the RN Reports, and listed in Voice Part Assignments.
If the Singer Position is removed by unchecking Singer, their Voice Part Overrides will be erased.
NT
 In addition to this setting, Note Takes must also have at least "Web Assistant" access to the website, as set in their WP User Profile.
 It is safe to add/remove Note Takers at any time.  Existing notes created by them will be unaffected.
 Note Taker
Dir
 This is only for Artistic and Guest Directors, who are the ones who create and decide on notes.  It does not include Associate Directors.  This setting should be made preseason, and not changed once Rehearsal Notes have been created.
 It is possible to add another Director, but Directors cannot be removed while there are any Notes assigned to them. If there are dependent notes, the Dir checkbox will be disabled.  In order to remove such a director, all the notes assigned to them must first be reassigned to another Director.

This same protection happens in the User's Profile Position.  It will be locked if the Director has Rehearsal Notes assigned.
 Artistic Director
Admin
 Administrators have additional capabilities beyond the other 3 roles.
 The position can be set/removed at any time.
 RN Admin
Voice Part assignments

The remainder of this form lists the songs and the default Voice Part Assignments based on the Singer's Primary Voice Part.  If the Primary Voice Part is changed, the default settings will automatically update, and you can see and adjust the results of the change.  (This is why the WP Users Profile's Primary Voice Part setting is disabled for active singers.  If changing the Primary Voice Part setting was done on the Profile page, one would not know that it altered the Voice Part Assignments.)

The default assignments can be overridden by selecting different or additional voice part(s) for any song.  These overrides do not change if the Primary Voice Part is changed.
