On Behalf of: Drop down
Only on: Ask Question for Directors
When a Note Taker needs to enter a question asked by another Singer, their name is entered here.  This is so that the Directors know who the question came from, the staff knows where to send an answer to the question if needed, and the Singer is able to use their TRACK QUESTIONS form to find its status.
EDITING A PUBLISHED NOTE, already copied (done > 0)
Only on: Edit Published Rehearsal Note (when Done count > 0)
This condition is indicated at the top of the form with:
You are editing a Published Note that has been read/copied.  These settings can no longer be changed...
If a Note has already been copied into scores, then only its content can be edited.  All of its "identifying information" must remain the same, so that Singers will know exactly which note has been changed.  The note's content should be rewritten in way that makes it clear:
  • It is a modification of an earlier marking.
  • What the previous marking was, and how it should now be modified or removed.
  • Consider adding [Modified] at the beginning of the content.
  • Consider using a strike-out font over the previous content that has changed.
When a Published Note is modified and Saved, a dialog will appear asking whether to clear everyone's "Done" status.  Normally you should click on "OK" to do this.  It returns the Note to a Singer's "My notes" list, and lets them know that this is a change they need to update in their scores.  However, there may be some cases where it is not necessary, for example:
  • The changes only enhance the description of the Note, but do not change what needs to be marked.
  • The Note was only reformatted for easier reading, and the marking to be made has not changed.

Sometimes changes are needed in a Published & marked Note's "identifying information":

Change How to handle it
Wrong Song
 Keep the original Note, mark it as "wrong song", optionally clear the "Done" settings.  Duplicate it and create a new Note with the correct song.
Wrong Start Measure
 Sometimes if its very close, it can be noted in the content, optionally clear the "Done" settings.  Otherwise, the safest thing to do is mark the error in this note, clear its "Done" settings, and duplicate a new note at the correct measure.
Adding a Voice Part
 This is a safe change, as the note is unchanged for the existing VPs, and adding the new VP will now add it to their My note filter.  An RN Admin can make this change.  The "Done" settings should not be cleared.
Removing a Voice Part
 This should normally not be done, unless it is obvious that the original parts were wrong (e.g. the VP to be removed is not even singing at that measure.)  The VPs should stay the same and the change should be noted in the content.  The "Done" settings should be cleared.
Changing Location
 This cannot be done, even if the Note needs to be removed.  Leave it in place, put in it's content that it's markings need to be removed from the score.  Clear the "Done" settings.
Changing Director
 This should never be needed for a Published Note.  The only purpose for this setting is for directing Questions to one of the Directors.
SONG DROP DOWN

This should always be set first.  It then establishes the range of allowed measures, the Section (Voice Parts) list, and the default Director.  If this is changed later, you need to recheck these settings.

MEASURE BOX
Enter the starting measure for the Q/N.  The possible range for this song is listed below the box and is used to validate the measure entered.
  • If the Q/N refers to the entire song, leave this blank. 
  • If the Q/N is for a range of measures (3-10), put the starting measure in this box, then indicate the range in the content. 
  • If this is a question for a set of measures (3,15,31) then put the first measure in the box and indicate the list of measures in the Question. 
    • Once the question has been answered, divide the Note into separate Notes for each measure using the Duplicate button.
Section

This shows a list of possible voice parts for the song.  The ones the Q/N pertains to should be highlighted in  blue .  Click on each to turn them on/off.

Director DROP DOWN

This defaults to the Director assigned to the Song in the Settings, Song List.  But it can be overridden if needed.  For example, sometimes questions about voice part assignments need to be redirected from a guest director to Allan.

Q/N Content Edit Box

This is where the contents of the Q/N are entered or edited.  The "Add Media" button should be avoided or only used for very small images that support the note.  The "Text" tab should only be used by those who know HTML.
