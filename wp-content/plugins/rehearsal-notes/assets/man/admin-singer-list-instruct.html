This list must be updated whenever Users are moved to/from the "Members" role, in other words, when the list of active Singers in the current Season changes.  This is done by clicking the "Import/Update" link above.  Re-importing will not erase per-song settings/overrides of Voice Parts.

Position (Pos) key: s - Singer, n - Note Taker, d - Director, a - RNote Admin.
The Primary Voice Part (PVP) is used to sort the singer lists on several pages across the website. It is also used as the default setting for voice parts assigned to each song. Voice Parts for each Singer/song can be set directly by clicking on them. All Singer settings can be edited by clicking on their name.  Changes are saved immediately.
