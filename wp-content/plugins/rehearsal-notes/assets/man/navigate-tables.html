Throughout the website there are tables formatted like this:

They all act in similar ways, and these instructions apply to most of them.
Header Rows
  • The upper header row contains the name of each column.
    • Clicking on most of them will sort that column
    • The sorted columns will have arrows indicating their current sort direction.
    • Use Shift-click to sort multiple columns at the same time
  • The lower header row contains filter boxes for filtering the rows
    • Normally these are like text search boxes
    • Some accept operators like: ! not, > greater than, < less than.
    • The filters always use AND logic.  The result of more than one filter is that each row must satisfy both conditions.
"Preset" drop-down

Some tables have a Preset drop-down at the top.  These provide different combinations of filters and sorting.

Hide Columns

This button enables hiding of columns to help in viewing larger tables.  Note: The button changes its name, depending on the state of hidden columns.

To hide columns:
  1. Click the  HIDE COLUMNS  button
    • A tiny " HIDE" checkbox will appear in each column header
  2. Check the columns you want to hide
  3. Click  HIDE SELECTED COLUMNS 
To change or show hidden columns:
  1. Click the  SHOW/HIDE COLUMNS  button
    • If the button only says  HIDE COLUMNS  it means there are currently no hidden columns
    • All hidden columns will now appear, along with their tiny checkboxes
  2. Check/Uncheck the columns you want to hide/show
  3. Click  HIDE SELECTED COLUMNS 
Download button

Some tables have a DOWNLOAD button.  It will download a CSV file that contains exactly what is showing on the screen.  Any filtering, sorting, and hidden columns will be applied to what is downloaded.

Email button

A few tables have an EMAIL button.  It will start an email addressed only to the persons/rows that are currently visible.
