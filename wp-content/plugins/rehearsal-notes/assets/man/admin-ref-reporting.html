Location: Admin -> Dashboard -> Settings -> Rehearsal Notes (third Tab)
Direct Link: https://www.consortchorale.org/wp-admin/options-general.php?page=rn-settings&tab=reports
This table is used to monitor the progress singers are making in getting Rehearsal Notes marked in their scores.  For more information on using this table see Navigating Common Tables.

The data is based upon:
  • The number of Rehearsal Notes each Singer has checked "Done"
  • Only notes for the Singer's Voice Part assignments (In their "My notes" filter) are counted
    • This is the reason why each singer will have a different total "Count"

Therefor, the "Count" column shows the number of assigned notes marked "Done" / the number of assigned notes for that Singer.

The "Last Online" column's timestamp will be in red if any published Rehearsal Notes have been modified more recently than when the Singer was last online.
