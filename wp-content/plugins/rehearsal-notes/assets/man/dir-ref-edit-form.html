Location: Admin -> Edit Rehearsal Notes, click on a Question
Direct Link: https://www.consortchorale.org/rehearsal-notes-admin/
This form enables a Director to answer a Question and on Save, sends it to the "For Review" location where it is reviewed by a Note Taker before publishing to the Singers.  Here are the parts of the form, starting from the top.

Song

Normally this should be left alone.  If it needs to be changed, make sure the Measure and Section settings are still ok.

Measure

This is the starting measure of the Question.  If it is blank, the question is meant for the entire song.  If the question refers to a range of measures (3-10) or a set of measures (3,16,41), then that should already be indicated in the Question's contents.

Section

This should already be set to the Voice Parts the Question is about.

Director drop down

You can use this Drop Down if the Question should be assigned to another Director.

Question Content Edit Box
This is where the question will be.  You can do one of:
  • Append your answer after the question
  • If the answer involves a marking in the music, replace the question with the instructions for the marking.

You do not need to be overly concerned about editing a clean answer.  Your reply will be reviewed by a Note Taker before being published to the Singers.  If the answer does not require a change in the music, the Note Taker will email your answer to the one who asked the question.

[seg-rnote-format]
Discussion Edit Box

This edit box is below the Question box.  This box is a private area only visible to the RN Staff and is never made public to the Singers.  It is where you can send comments or questions back to the Note Takers, particularly if you need more information about the question.
