This form enables Singers to send questions to the Directors.  It makes it easier for the question to be answered and become a Rehearsal Note because the Song, (Start) Measure, and (Voice Parts) Section gets filled in from the beginning.
Song
Select which song the question is about.  Choose this first, as it then sets up the MEASURE and SECTION options according to what the song contains.
Measure
This is the starting measure for the question.  Only put one measure number in the box.  If the question involves a range of measures (3-8) or a set of measures (3,16,22) put only the first measure in the box and indicate the rest in the question.  If the question applies to the entire song, leave this blank.
Section
These are the Voice Parts that the question affects.  The selected voice parts are highlighted in  blue .  Click each to turn off/on.
Question
Enter your question in the edit box.  Normally use the edit box with the "Visual" tab.  The "Text" tab is only for those familiar with HTML.   [seg-rnote-format]
Send
Click when done.  Your question will then go to the Note Taker's Inbox.  One of the Note Takers will review it and send it to the Directors.  You can follow its progress by clicking the TRACK QUESTIONS button on the Rehearsal Notes page.  If you need to make any changes to your question, in the Track Questions form you will find email links to contact the Rehearsal Notes staff.
