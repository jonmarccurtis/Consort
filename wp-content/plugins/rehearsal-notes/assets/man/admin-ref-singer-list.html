Who appears on this list is controlled by WP Roles: anyone who is Singer, Web Assistant, or Board.  The list is kept separately and must be synchronized whenever WP Roles are changes (people are registered or drop out of the current Season).  This is done using the "Import/Update Singer List" link at the top of the page.  It is safe to do this at any time as the WP user changes are merged with the existing list and the results of the merge are reported at the top of the page.

The Singer List is a standard table, and more information on how to use it can be found in Navigating Common Tables.  More information about how to use this page can be found in Instructions for Administrators.

Here are details about each column in the table:

F & LAST

First initial and Last name of each person active in the current Season.  When up-to-date, this list includes all those with a WP Role of Singer, Web Assistant, or Board.

Pos
This indicates the RN Position of each person.  The Key is:
s - Singer
n - Note Taker
d - Director
a - RN Administrator
Details about each position and how to change them safely can be found in the Singer Settings form.
PVP

Singer's Primary Voice Part.  It is used for setting the default Voice Part Assignments for each song. 

SONG COLUMNS

The remaining columns contain the songs from the Song List tab and show each Singer's assigned Voice Part(s) for that song.  It makes this table rather wide, so the  HIDE COLUMNS  button can be useful for reducing the width and making this table manageable.

Editing a Singer's Settings

Clicking on a Singer's Name, Pos, or PVP will bring up the Singer Settings form for the editing of Positions, Primary Voice Part, and Voice Part Assignments for each song.

Clicking on a single Voice Part cell in one of the Song columns bring up a quick-edit form to modify that one assignment.

There is no "Save" button on this page.  All changes are immediately saved when they are made.
