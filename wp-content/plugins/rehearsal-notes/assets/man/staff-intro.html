Rehearsal Notes provide an online way to record and distribute changes to be marked in our music scores, requested by the Directors.  It also includes a way for singers to ask questions outside of rehearsals. It is located in a variety of places on the website:
Menu Path
 Contents
Singer Info -> Rehearsal Notes
 Published notes for Singers to mark in their scores.  It also contains links to the following pages and forms:
  • Ask Question form.
  • Track Questions asked
  • See and override Voice Part assignments for "My Notes" filter
  • PRINT - open a set of notes for printing and marking "Done"
Singer Info -> Voice Part Assignments
 Complete list of singer's voice part assignments given by the Directors.
[Name] -> Website help
 Singer's Instructions under "How to use the Rehearsal Notes Page?"
Admin -> Edit Rehearsal Notes
(for Note Takers and Directors)
 Used by RN Staff to add, edit, review, and publish notes.  And to process questions.  Notes are kept in a single table with multiple views, called "Locations":
  • Note Taker's Inbox
  • Director's Inbox
  • Preparation and Review
  • Published
  • Trash
It also has several forms:
  • Add/Edit Notes/Questions (for Note Takers and Directors)
  • Note/Question History, Log
Admin -> Dashboard -> Settings -> Rehearsal Notes
 Contains Tabs:
  • Song List - sets names, length, and voice parts for each song
  • Singer List - defines RN roles and Voice Parts for each song
  • Reporting - tracks how well the singers are getting their music marked
  • Administration - clean out and reset data at the end of the season
  • Help - Administrator's Instruction manual

Users of the Rehearsal Notes System are categorized into 4 roles.  The following table shows what each role does, and where its Instructions are located on the website.  These instructions are also shown in the next section of this manual.
