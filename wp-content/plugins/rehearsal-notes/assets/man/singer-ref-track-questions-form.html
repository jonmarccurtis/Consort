This form shows a complete log of questions you have asked.  Each version of the question is shown as it progresses through the process. 
The typical progression is:
  1. Note Taker's Inbox (Reviews, check for duplicates, already answered, etc)
  2. Director's Inbox (Answers the question)
  3. For Review (Final check before sending out to the Singers)
  4. Published (Shows up as a Rehearsal Note)

Some questions can be answered without needing to create a Rehearsal Note.  In this case, you should get the answer via an email and the question will be sent to Trash with the explanation that it has been answered.

If you need to modify your question or want to contact the staff about your question, click on the most recent Editor's name to start an email to them.
