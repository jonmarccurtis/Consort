This topic only covers the parts of the page needed by Directors to answer questions in their Inbox.  For a complete explanation of all items on this page, see Note Taker's Edit Rehearsal Notes Page.

Presets Drop Down

This should be set to either "Director's Inbox" or "- [Your Name]'s Inbox".  In this setting, the only items you should see in the table are questions to be answered.

ADD NOTE BUTTON

The ADD NOTE button can be used to enter Rehearsal Notes directly. If you wish to do this, instructions can be found in Note Taker's Add Note Form and Note Taker's Instructions.

Instructions for Directors

At the top left of the table is a link: Instructions for Directors.  This will take you to Instructions for Directors.  In the upper right corner is another help link: ?. This is a link back to this topic.

ANSWERING A QUESTION

Click on a row/question in the table to bring up the Director's Edit Question form, used for answering questions.
